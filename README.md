# Three.js Journey - Ejercicios

Este repositorio contiene los ejercicios y proyectos del curso [Three.js Journey](https://threejs-journey.com/) de Bruno Simon.

## Descripción

A través de estos ejercicios se exploran los conceptos fundamentales de Three.js, incluyendo:

- Transformación de objetos
- Animaciones
- Cámaras
- Geometrías
- Texturas y materiales
- Texto 3D
- Y más...

Cada carpeta representa una lección del curso con su propio proyecto y ejercicios prácticos.

## Estructura

- `04-transform-objects/` - Transformación de objetos
- `05-animations/` - Animaciones
- `06-camera/` - Cámaras
- `07-fullscreen-and-resizing/` - Pantalla completa y redimensionamiento
- `08-Geometries/` - Geometrías
- `09-debug-ui/` - Interfaz de depuración
- `10-textures/` - Texturas
- `11-materials/` - Materiales
- `12-3d-text/` - Texto 3D
- `first-proyect/` - Primer proyecto

## Instalación

Cada proyecto tiene sus propias dependencias. Para instalar las dependencias de un proyecto específico:

```bash
cd nombre-del-proyecto
npm install
```

## Ejecución

Para ejecutar un proyecto:

```bash
cd nombre-del-proyecto
npm run dev
```
